* Rinku (リンク) /rin-ku/ [Japanese: [[https://jisho.org/search/%E3%83%AA%E3%83%B3%E3%82%AF][link]]]

A macOS command-line utility for fetching link previews and metadata from URLs:

- Extract metadata (title, URL, image) from web links.
- Render visual link previews as PNG images.
- Cache results for faster subsequent lookups.
- Return details as JSON.

* Usage

** Render preview image

#+begin_src sh
  rinku --render https://soundcloud.com/shehackedyou/glaceux
#+end_src

Returns:

#+begin_src json
  {
    "preview": "path/to/preview.png"
  }
#+end_src

#+HTML: <a href="https://soundcloud.com/shehackedyou/glaceux"><img src="https://github.com/xenodium/rinku/blob/main/render.png?raw=true" width="200px" alt="Rendered preview of SoundCloud link" /></a>

** Fetch metadata

#+begin_src sh
  rinku https://soundcloud.com/shehackedyou
#+end_src

Returns:

#+begin_src json
  {
    "title": "glaceux",
    "url": "https://soundcloud.com/shehackedyou",
    "image": "path/to/preview.png"
  }
#+end_src
